#### Summary

**[[Protein dynamics|Protein structural ensembles]] can be sampled with [[alphafold3|AlphaFold3]] and related methods to match experimental data like [[X-ray-crystallography|X-ray]] or [[cryo-EM|EM]] density, or [[NMR]] NOEs, by using [[diffusion-guidance|guidance sampling]]** [@maddipatla2025]. AF3 by default predicts tighter conformational ensembles.
