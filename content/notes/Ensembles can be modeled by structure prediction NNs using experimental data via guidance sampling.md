#### Summary

**[[Protein dynamics|Protein structural ensembles]] can be sampled with [[AlphaFold|AF3]] and related methods to match experimental data like [[X-ray-crystallography|X-ray]] or [[cryo-EM|EM]] density, or [[NMR]] NOEs, by using [[Diffusion guidance|guidance sampling]]** [^maddipatla2025]. AF3 by default predicts tighter conformational ensembles.

[^maddipatla2025]: Maddipatla et al. (2025) "Inverse problems with experiment-guided AlphaFold." https://doi.org/10.48550/arxiv.2502.09372
