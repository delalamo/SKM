#### Summary
**Proteins designed using PLMs more unique than those designed using Potts models** [^madani2023]. For example, $\beta$-lactamase sequences designed using [[Potts models]] by [^fram2023] were found to be active up to 50% sequence identity, but not more. In contrast, lysozyme sequences designed using (ProGen) model after fine-tuning achieve activity with identities as low as 31% [^madani2023]. In the same experiment, bmDCA-generated sequences were inactive.
