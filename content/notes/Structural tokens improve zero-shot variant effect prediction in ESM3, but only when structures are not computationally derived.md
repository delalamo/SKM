#### Summary

**[[Protein structure tokenization|Structural tokens]] improve zero-shot [[Variant effect prediction|variant effect prediction]] in [[ESM|ESM3]], but only when using the experimental structure of the WT** (Loux et al 2024[^loux2024]). The use of computational models obtained using [[Rosetta]] or short [[MD simulations]]  substantially lowers the Spearman correlation, despite the resulting embeddings being almost identical.
