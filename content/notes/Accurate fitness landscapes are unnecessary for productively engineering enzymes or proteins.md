#### Summary
**Accurate fitness landscapes are unnecessary for productively engineering enzymes or proteins.** This has been true empirically when generating sequences using random techniques such as error-prone PCR, but [^rapp2023] demonstrated that autonomous laboratories that design sequences using [[Gaussian processes]] can iteratively improve fitness even in stages where the algorithm is not modeling landscapes accurately.
