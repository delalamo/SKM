#### Summary
The **Chapman-Kolmogorov test** is used to evaluate the appropriateness of a specific lag time when building [[Markov State Models|Markov state models]] and when running [[Time-lagged independent component analysis|time-lagged independent component analysis]]. It states that an appropriate lag time $\tau$ when building a transition matrix $\mathbf{P}$ should meet the condition $\mathbf{P}({n\tau}) = \left(\mathbf{P}(\tau)\right)^{n}$.
