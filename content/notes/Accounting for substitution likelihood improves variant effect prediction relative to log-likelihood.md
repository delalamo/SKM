#### Summary
**Accounting for substitution likelihood (e.g., via the codon table) improves [[Variant effect prediction|variant effect prediction]] relative to just using pure log-likelihood** [^lu2026]. This was done in DASM [^matsen2026] using neutral evolution rates inferred from untranscribed portions of the genome.
