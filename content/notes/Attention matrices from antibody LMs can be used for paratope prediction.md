#### Summary
**[[Attention (machine learning)|Attention]] matrices calculated from [[Antibody language models|antibody LMs]] can be used for paratope prediction** (Ruffolo et al 2021[^ruffolo2021]). This was done using AntiBERTy, and authors attribute this to the use of multiple instance learning, which teaches the model to discriminate between binding and non-binding sequences via overrepresentation of binders in [[Immune repertoires]].
