#### Summary
**[[Attention (machine learning)|Attention]] matrices calculated from [[Antibodies|antibody]] [[Protein language models|PLMs]] can be used for paratope prediction** [@ruffolo2021]. This was done using AntiBERTy, and authors attribute this to the use of multiple instance learning, which teaches the model to discriminate between binding and non-binding sequences via overrepresentation of binders in [[Immune repertoires]].
